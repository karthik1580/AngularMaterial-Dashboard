----------------------------------------------------------------------------------------------------

1, ng new dashboard
2, Add the project in .code
3, Install angular material [ng add @angular/material]
4, install highchart packages[npm i highcharts-angular --save] [npm install highcharts --save]
5, install Flex Layout package [npm i @angular/flex-layout @angular/cdk --save]
6, create material model and include in app-module [ng g m app-material]
----------------------------------------------------------------------------------------------------

7, Create the first layout [ng g c layout/default, ng g m layout/default]
8, Create a component for the Dashboard page
9, Use the first layout as a Parent Route Component in app-routing and the Dashboard page as child component
